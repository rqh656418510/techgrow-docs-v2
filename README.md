## TechGrow 开放平台官方文档

### 依赖环境

- [Node.js v14+](https://nodejs.org/en/)
- [VuePress v1.x](https://v2.vuepress.vuejs.org/zh/)
- [Yarn v1 classic](https://classic.yarnpkg.com/zh-Hans/)

### 项目构建

``` sh
# 全局安装Yarn
$ npm i yarn -g

# 安装项目的默认依赖
$ yarn install

# 本地启动文档预览服务
$ yarn docs:dev

# 构建生成静态文件（可用于部署Web服务器）
$ yarn docs:build
```
